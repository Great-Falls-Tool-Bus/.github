# Great Falls Tool Bus

**A shared tool library on wheels — Lewiston-Auburn, on the Androscoggin, at the Great Falls.**

*private during buildout — public at launch*

</div>

---

Great Falls Tool Bus is a mobile tool-lending library serving Lewiston-Auburn: check
out real tools, learn to use them, bring them back. Named for the falls on the
Androscoggin that anchor the twin cities — the bus meets people where they are
instead of asking them to come to a fixed location.
